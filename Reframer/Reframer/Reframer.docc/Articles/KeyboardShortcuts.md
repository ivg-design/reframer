# Keyboard Shortcuts

Complete reference for all keyboard controls.

## Overview

Reframer is designed for keyboard-driven workflows. Arrow keys pan the video when unlocked, while frame stepping and zoom use scroll wheel or input fields.

## File

| Action | Shortcut |
|--------|----------|
| Open video | **⌘ O** |

## Playback

| Action | Shortcut |
|--------|----------|
| Play / Pause | **Space** |

Frame stepping is available via global shortcuts or scroll wheel.

## Pan (Arrow Keys)

When unlocked, arrow keys pan the video:

| Action | Shortcut |
|--------|----------|
| Pan 1 pixel | **← → ↑ ↓** |
| Pan 10 pixels | **⇧ + arrows** |
| Pan 100 pixels | **⇧⌘ + arrows** |

## View

| Action | Shortcut |
|--------|----------|
| Reset to 100% | **0** |
| Reset zoom and pan | **R** |

Zoom is controlled via scroll wheel or input field arrows when focused.

## Window

| Action | Shortcut |
|--------|----------|
| Toggle lock | **L** |
| Show help | **H** or **?** |
| Close help/panel | **Esc** |

## Filters

| Action | Shortcut |
|--------|----------|
| Toggle filter panel | **F** |
| Close filter panel | **Esc** |

## Input Fields

When a numeric field (frame, zoom, opacity) is **focused**:

| Action | Shortcut |
|--------|----------|
| Increment by 1 | **↑** |
| Decrement by 1 | **↓** |
| Increment by 10 | **Shift ↑** |
| Decrement by 10 | **Shift ↓** |
| Increment by 10 (Option/Ctrl) | **Option ↑** / **Ctrl ↑** |
| Decrement by 10 (Option/Ctrl) | **Option ↓** / **Ctrl ↓** |
| Fine increment (zoom only) | **⌘ ↑** |
| Select all | **⌘ A** |
| Apply and defocus | **Enter** or **Esc** |

> Note: Arrow keys only adjust values when an input field is focused. Outside input fields, arrow keys pan the video.

## Global Shortcuts (Lock Mode)

These work when Reframer is locked, even if it isn't the active app:

| Action | Shortcut |
|--------|----------|
| Toggle lock | **⌘ Shift L** |
| Previous frame | **⌘ Page Up** |
| Next frame | **⌘ Page Down** |
| Back 10 frames | **⌘ Shift Page Up** |
| Forward 10 frames | **⌘ Shift Page Down** |

## Mouse Controls

When the window is unlocked:

| Action | Gesture |
|--------|---------|
| Pan video | **Click + Drag** on video |
| Step frames | **Scroll** on video |
| Zoom (5%) | **Shift + Scroll** |
| Fine zoom (0.1%) | **⌘ Shift + Scroll** |
| Move window | **Drag** control bar or handle |
| Resize window | **Drag** window edges |
