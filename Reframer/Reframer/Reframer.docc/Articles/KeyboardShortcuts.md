# Keyboard Shortcuts

Complete reference for all keyboard controls.

## Overview

Reframer is designed for keyboard-driven workflows. Most actions have keyboard shortcuts, and some work globally even when another app is active.

## File

| Action | Shortcut |
|--------|----------|
| Open video | **⌘ O** |

## Playback

| Action | Shortcut |
|--------|----------|
| Play / Pause | **Space** |
| Previous frame | **←** |
| Next frame | **→** |
| Back 10 frames | **Shift ←** |
| Forward 10 frames | **Shift →** |

## Zoom

| Action | Shortcut |
|--------|----------|
| Zoom in (5%) | **↑** or **+** |
| Zoom out (5%) | **↓** or **-** |
| Zoom in (10%) | **Shift ↑** |
| Zoom out (10%) | **Shift ↓** |
| Reset to 100% | **0** |
| Reset zoom and pan | **R** |

## Window

| Action | Shortcut |
|--------|----------|
| Toggle lock | **L** |
| Show help | **H** or **?** |
| Close help | **Esc** |

## Input Fields

When a numeric field (frame, zoom, opacity) is focused:

| Action | Shortcut |
|--------|----------|
| Increment by 1 | **↑** |
| Decrement by 1 | **↓** |
| Increment by 10 | **Shift ↑** |
| Decrement by 10 | **Shift ↓** |
| Fine increment (zoom only) | **⌘ ↑** |
| Fine decrement (zoom only) | **⌘ ↓** |
| Apply and defocus | **Enter** or **Esc** |

## Global Shortcuts

These work even when Reframer is not the active app:

| Action | Shortcut |
|--------|----------|
| Toggle lock | **⌘ Shift L** |
| Previous frame | **⌘ Page Up** |
| Next frame | **⌘ Page Down** |
| Back 10 frames | **⌘ Shift Page Up** |
| Forward 10 frames | **⌘ Shift Page Down** |

## Mouse Controls

When the window is unlocked:

| Action | Gesture |
|--------|---------|
| Pan video | **Click + Drag** on video |
| Step frames | **Scroll** on video |
| Zoom (5%) | **Shift + Scroll** |
| Fine zoom (0.1%) | **⌘ Shift + Scroll** |
| Move window | **Drag** control bar or handle |
| Resize window | **Drag** window edges |

## Focus Return

When you press **Enter** or **Esc** in an input field, focus returns to the app you were using before Reframer. This lets you quickly adjust a value and continue working without manual app switching.
