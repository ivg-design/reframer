Adjusting Opacity

Slider

Drag the opacity slider in the control bar. The slider ranges from nearly transparent (2%) to fully opaque (100%).

Numeric Input

  1. Click the opacity percentage field
  2. Type a value from 2 to 100
  3. Press Enter

Arrow keys while focused:

  • ↑ / ↓ — Step by 1%
  • ⇧↑ / ⇧↓ — Step by 10%

Opacity Range

ValueEffect
2%Nearly invisible, just a hint
25%Very transparent, easy to see through
50%Balanced blend
75%Video dominant
100%Fully opaque

Use Cases

  • Tracing — Set 30-50% to draw over the video
  • Comparison — Set 50% to see both equally
  • Quick reference — Set 80-100% for clarity
  • Background guide — Set 15-25% for subtle reference
