Video Filters

Apply real-time effects to enhance your reference material

Overview

Reframer includes a variety of video filters that can be applied in real-time. Unlike traditional single-filter applications, Reframer allows you to combine multiple filters to create exactly the effect you need.

Available Filters

FilterDescription
EdgesHighlights edges, useful for tracing outlines
SharpenIncreases image sharpness and detail
Unsharp MaskProfessional sharpening with radius and intensity
High ContrastIncreases contrast for better visibility
SaturationAdjust color intensity (0 = grayscale)
MonochromeConverts to sepia/monochrome tones
InvertInverts all colors
Line OverlayCreates a line-art effect
NoirBlack and white film effect
ExposureAdjust brightness (−EV darker, +EV brighter)

Using Filters

Quick Access Menu

  1. Hold the filter button (checkerboard icon) for 300ms
  2. A menu appears with all available filters
  3. Toggle individual filters using the checkmarks
  4. Multiple filters can be active simultaneously

Filter Settings Panel

  1. Click the filter button to open the settings panel
  2. Use checkboxes to enable/disable filters
  3. Adjust filter parameters using the sliders

Keyboard: Press F to toggle the filter panel, Esc to close.

Menu Bar

Access filters through the Filter menu:

  • Toggle individual filters
  • Clear All Filters
  • Open Filter Settings panel
  • Reset filter parameters to defaults

Combining Filters

Filters are applied in a consistent order when combined. Useful combinations:

  • Edges + High Contrast — Creates bold, visible outlines
  • Sharpen + Saturation (0) — Crisp grayscale reference
  • Unsharp Mask + Exposure (+1) — Enhances detail in dark footage
  • Noir + Exposure (−0.5) — Dramatic dark B&W

Tips

  • The filter button turns blue when any filter is active
  • Hover over the filter button to see which filters are applied
  • Use Clear All to quickly disable all filters
  • Filter parameter settings are independent of which filters are active
